# Smart Task Manager

Enterprise-style full stack task management system built using Spring Boot, Angular, PostgreSQL, JWT Authentication, and Docker.

## Features
- JWT Authentication
- Role-Based Access
- Task CRUD Operations
- Dashboard
- File Upload
- Pagination & Filtering
- Docker Support

## Tech Stack
### Backend
- Java
- Spring Boot
- PostgreSQL

### Frontend
- Angular

### DevOps
- Docker
- GitHub
